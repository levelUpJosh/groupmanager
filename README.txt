NEA program for managing groups

Program must be run through Django: https://www.djangoproject.com/download/

To run the program after installing Django, while holding shift right click on white space in the folder and select "Open Powershell Window Here" (Command Prompt will also work). Run the command "py manage.py runserver" and a few seconds later, the program will run and the website will open at the URL "Http://127.0.0.1:8000"
